Accept payment using SOLANA 


# To start 
Install dependencies
cd solana-pay/point-of-sale

yarn install
# start local server 
yarn dev
# In a separate terminal, run a local SSL proxy

open "https://localhost:3001?recipient=Your+Merchant+Address&label=Your+Store+Name"
